Supported Communication Protocols

This page is intended to serve as index for communication protocols that are supported by FORTE.

  • FBDK/IP: Protocol defined in the IEC 61499 Compliance Profile for Feasibility Demonstrations and default protocol in FORTE.
  • MQTT
  • OPC UA
  • OPC DA
  • Modbus
  • openPOWERLINK
  • Communication using Matlab
